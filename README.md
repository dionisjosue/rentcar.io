Política de Privacidad

Esta aplicación utiliza WhatsApp Business API únicamente para
comunicación relacionada con la renta de vehículos.

No almacenamos información personal sensible fuera de lo necesario
para la gestión de reservas, contratos y pagos.

Los datos compartidos por el usuario a través de WhatsApp se usan
exclusivamente para:
- Consultar disponibilidad de vehículos
- Gestionar reservas
- Enviar recordatorios y confirmaciones

No compartimos información con terceros.
El usuario puede solicitar la eliminación de sus datos en cualquier momento.
