Política de Privacidad

Esta aplicación utiliza WhatsApp Business Cloud API para automatizar el proceso de renta de vehículos.

La información proporcionada por los usuarios se utiliza únicamente para:

  • Consulta de disponibilidad de vehículos
  • Gestión de reservas
  • Generación de contratos
  • Procesamiento de pagos
  • Envío de recordatorios

No compartimos información personal con terceros. Los datos se almacenan de forma segura y solo durante el tiempo necesario.

El usuario puede solicitar la eliminación de sus datos en cualquier momento.
